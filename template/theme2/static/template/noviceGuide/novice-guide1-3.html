新手中级教程
  • 为了方便理解交易中的收费，首先你需要 知道交易的全过程：
    建仓

    新买入或卖出一定数量的黄金白银

    持仓

    持有这笔买入或卖出的黄金白银

    平仓

    针对持仓方向做相反的同等数量的买卖， 以了结此笔黄金白银的交易

    交易全过程中会收取哪些费用？
    在当日平仓的情况下，只收取点差，若非当日平仓，会按日收取隔夜利息

    金道贵金属无其他任何交易手续费

  • 点差费用
    点差是在任何时点买入价和卖出价的价差，相当于股票交易中券商所收取的手续费；伦敦金点差为0.5美元/盎司、伦敦银点差为0.04美元/盎司
    计算公式
    （买入价-卖出价）x 合约单位 x 手数 = 点差费用 （1247.30-1246.80）x 100 x 1 = 50
    手：

    手数是交易买卖单位，最小买卖单位为0.01手

    合约单位：

    合约单位是指交易1手伦敦金/银合约的数量。一手伦敦金的合约单位为100盎司，一手伦敦银的合约单位为5000盎司。

    盎司：

    是国际计量单位，相当于千克、克、两的说法

  • 隔夜利息费
    隔夜利息是持仓过夜（超过当日结算时间）所产 生的利息费用
    结算时间
    夏令时

    星期二到星期五每日凌晨05:00，周六凌 晨04:45

    冬令时

    星期二到星期五每日凌晨06:00，周六凌 晨05:45

    计算公式
    隔夜利息=建仓价x合约单位x手数x利率x(日数/360)

    账户级别越高利率、点差优惠越大哦！

    若持仓跨越周末的休市，会收取周五、周六 和周日三天利息。
  • 交易纯利润
    (卖出价－买入价)×合约单位×手数－隔夜利息 =交易纯利润
    测试
    1000美元总资金，1200美元时买入1手伦敦金，在持仓次日于1205美元/盎司的价格全部平仓，隔夜利息是多少？交易纯利润多少？
    测试答案
    隔夜利息 = 1200 x 100 x 1 x 1.45% x 1/360 ＝ 4.83美元
    交易纯利润=（1205-1200）x 100 x 1- 4.83 ＝ 495.17美元
    立即开户
